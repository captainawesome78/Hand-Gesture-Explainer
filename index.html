How to use this in real video calls (Zoom, Meet, Teams)

1. Open this page in Chrome/Edge and start the camera.

2. Draw using mouse, hand gestures (pinch thumb and index to draw), or add shapes with voice.

3. Option A (easiest): Share your entire screen or browser tab in the call.

4. Option B (pro): Use OBS Studio + Virtual Camera plugin. Add this browser window as a source and pipe the output as your webcam.

Future version: Chrome extension that injects floating overlay directly into Zoom/Meet tabs.
